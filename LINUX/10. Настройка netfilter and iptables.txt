ptables является утилитой, выполняющей функции межсетевого экрана. 
Ее настройка производится в командной строке, с помощью правил iptables 
можно разрешать или блокировать прохождение трафика. Когда происходит 
попытка установления соединения с текущей машиной, iptables просматривает 
список правил в списке, чтобы понять, 
как нужно поступить в этом случае. Если правила нет, то выполняется 
действие по умолчанию.

sudo apt-get install iptables

Правило ниже позволяет заблокировать все соединения с IP-адреса 10.10.10.10:

iptables -A INPUT -s 10.10.10.10 -j DROP

Правило ниже позволяет заблокировать соединения из диапазона IP-адресов 10.10.10.0/24

iptables -A INPUT -s 10.10.10.0/24 -j DROP

Ниже показано, как заблокировать SSH-соединения с хоста 10.10.10.10:

iptables -A INPUT -p tcp --dport ssh -s 10.10.10.10 -j DROP

Вот так можно заблокировать SSH-соединения с любого IP-адреса.

iptables -A INPUT -p tcp --dport ssh -j DROP
